# 跨性别相关医疗照护

## 精神科的诊断

正式接受性别肯定医疗之前，需要进行跨性别身份的诊断，<mark>这类诊断需由三级甲等医院的精神科医师或精神专科医院开具，但并非所有的精神科医师都具备诊断跨性别身份的临床经验</mark>。诊断主要通过沟通（说明既往及当前状态和就诊诉求）和量表评估（如 SCL-90、MMPI）进行，有时可能会涉及激素、染色体等检查。诊断证明的开具流程因地区、医院或医师而异，通常需要至少 1 年的观察期。

> [!NOTE] 根据各地医院应用的诊断标准不同，其会包含的诊断名称有：
> 
> * 易性症 / 性身份障碍（ICD-10）
> * 性别不安 / 性别烦躁（DSM-5）
> * 性别不一致（ICD-11）

上述诊断在临床上具备相同效力，均可用于进行性别肯定激素治疗及性别肯定手术。需要注意的是，虽然跨性别身份的诊断需由精神科医师进行，但<mark>跨性别本身并不是精神疾病</mark>。

## 青春期阻滞治疗

青春期阻滞治疗适用于青春期的跨性别青少年，通过促性腺激素释放激素类似物（GnRHa）抑制性腺发育和激素分泌，从而延缓青春期的发育。<mark>青春期阻滞治疗旨在暂停指派性别的第二性征发展，缓解性别焦虑，并为跨性别青少年提供探索性别身份的时间</mark>。

> [!NOTE] 国内目前的跨性别青春期阻滞治疗资源参考：
> 
> * 北京大学第三医院易性症综合 / 激素治疗门诊
> * 复旦大学附属儿科医院跨性别 MDT 门诊
> * 暨南大学附属第一医院跨性别序列医疗 / 内分泌治疗门诊

> [!NOTE] 在国内进行青春期阻滞治疗的基本条件：
> 
> * 未患有影响自我认知判断的精神疾病，或原有精神疾病已得到有效控制
> * 对自身情况具有清晰的认知，有明确的进行青春期阻滞治疗的需求
> * 经相关精神科医师评估，获取跨性别身份的相关诊断
> * 需由监护人同意进行青春期阻滞治疗
> * 性发育达到 Tanner 2 期（青春期早期，出现初步的第二性征发育）或以上

## 性别肯定激素治疗

性别肯定激素治疗包括女性化激素治疗和男性化激素治疗，分别适用于女性倾向跨性别者和男性倾向跨性别者。<mark>性别肯定激素治疗旨在暂停指派性别的第二性征发展，促进认同性别的第二性征发育，从而提升自我接纳、缓解性别焦虑并改善心理健康。对于术后跨性别者则是维持对应性别的激素水平范围</mark>。激素治疗方案因人而异，<mark>应根据实际情况和个人需求制定</mark>。

> [!NOTE] 国内目前的跨性别激素治疗资源参考：
> 
> * 北京大学第三医院易性症综合 / 激素治疗门诊
> * 上海交通大学医学院附属第九人民医院性别医学专科门诊（南部院区）
> * 暨南大学附属第一医院跨性别序列医疗 / 内分泌治疗门诊

> [!NOTE] 在国内进行性别肯定激素治疗的基本条件：
> 
> * 未患有影响自我认知判断的精神疾病，或原有精神疾病已得到有效控制
> * 对自身情况具有清晰的认知，有明确的进行激素治疗的需求
> * 经相关精神科医师评估，获取跨性别身份的相关诊断
> * 治疗前检查无激素治疗相关禁忌症
> * 需由监护人同意进行激素治疗

> [!NOTE] 应用性别肯定激素治疗的前置检查项目：
> 
> * 血常规
> * 电解质
> * 骨密度
> * 凝血功能
> * 肝肾功能
> * 血糖血脂
> * 乳腺超声
> * 性激素 6 项（泌乳素、卵泡刺激素、黄体生成素、雌二醇、睾酮、孕酮）

需要注意的是，不同医院的前置检查项目可能并不相同。<mark>选接受激素治疗后，建议每 3 个月复查肝肾功能、雌二醇、睾酮、泌乳素水平（男性倾向跨性别者无需此项），并每年进行一次全面体检</mark>。下文所示的激素治疗方案、效果及风险来自跨性别和多元性别人群健康照护指南（第八版）[^1]，不构成专业医疗建议，具体用药方案需根据激素水平动态调整。

> [!NOTE] 性别肯定激素治疗的一般方案：
> 
> * 女性化激素治疗（使用如醋酸环丙孕酮等抗雄激素和雌二醇等雌激素）
> * 男性化激素疗法（使用如睾酮等雄激素）

> [!NOTE] 女性化激素治疗的用药方案（以雌激素为基础的方案）<sup>*</sup>：
> 
> 雌激素：
> * 口服 / 含服药物：2～6 毫克 / 每天
> * 经皮吸收药物：0.025～0.2 毫克 / 每天
> * 肌肉注射药物：2～10 毫克 / 每周、5～30 毫克 / 每 2 周
> 
> 抗雄激素：
> * 螺内酯：100～300 毫克 / 每天
> * 醋酸环丙孕酮：10 毫克 / 每天
> 
> *目前无充分证据表明使用黄体酮等孕激素能促使身体女性化发育，且潜在获益远超其风险，虽然醋酸环丙孕酮同样为孕激素，但其在激素治疗中主要作为抗雄使用，在作用机制上存在明显区别。

> [!NOTE] 男性化激素治疗的用药方案（以雄激素为基础的方案）：
> 
> 雄激素：
> * 庚酸睾酮 / 环戊丙酸睾酮：每周肌肉或皮下注射 50～100 毫克、每两周肌肉注射 100～200 毫克
> * 十一酸睾酮：每 12 周肌肉注射 1000 毫克、每 10 周肌肉注射 750 毫克
> * 睾酮凝胶：50～100 毫克 / 每天

> [!NOTE] 性别肯定激素治疗的激素监测参考：
> 
> 女性倾向跨性别者：
> * 血清雌二醇水平应在 100～200 pg/mL 范围内
> * 血清睾酮水平应低于 50 ng/dL
> 
> 男性倾向跨性别者：
> * 血清睾酮水平应在 400～700 ng/dL 范围内

> [!NOTE] 女性化激素治疗的效果时间：
> 
> |效果|开始时间|最大作用时间|
> |:-:|:-:|:-:|
> |身体脂肪重新分布|3～6 个月|2～5 年|
> |体能下降|3～6 个月|1～2 年|
> |皮肤出油减少|3～6 个月|未知|
> |面部和躯体毛发减少|6～12 个月|>3 年|
> |胸部发育|3～6 个月|2～5 年|
> |性欲下降|1～3 个月|未知|

> [!NOTE] 女性化激素治疗的相关风险：
> 
> |风险程度||
> |:-:|:-:|
> |很可能增加|肝脏疾病风险<br>静脉血栓栓塞<br>高钾血症<sup>S</sup><br>甘油三酯增加<br>体重增加|
> |在存在额外风险因素的情况下很可能增加|心血管疾病<br>脑血管疾病<br>脑膜瘤<sup>C</sup><br>多尿/脱水<sup>S</sup><br>胆石症|

> [!NOTE] 男性化激素治疗的效果时间：
> 
> |效果|开始时间|最大作用时间|
> |:-:|:-:|:-:|

> |体毛增多|6～12 个月|>5 年|
> |雄激素性脱发|6～12 个月|>5 年|

> |第一性征男性化|1～6 个月|1～2 年|

> 
> 严格戒烟、减少饮酒、控制高油高脂饮食、保持适度运动、避免久坐、按医嘱补充维生素。

## 性别肯定手术

性别肯定手术包括男跨女性别肯定手术和女跨男性别肯定手术，分别适用于女性倾向跨性别者和男性倾向跨性别者。<mark>性别肯定手术旨在减少指派性别的部分性征，促进认同性别的性征表达，从而提升自我接纳、缓解性别焦虑并改善心理健康</mark>。

> [!NOTE] 国内目前的性别肯定手术医疗资源参考：
> 
> * 中国医学科学院整形外科医院
> * 北京大学第三医院整形外科
> * 上海交通大学医学院附属第九人民医院整复外科
> * 中国人民解放军第四一一医院整形外科

> [!NOTE] 在国内进行性别肯定手术的基本条件：
> 
> * 年满 18 周岁
> * 术前检查无手术相关禁忌症
> * 经相关精神科医师评估，获取跨性别身份的相关诊断
> * 对自身情况具有清晰的认知，有明确的进行手术的需求
> * 手术对象本人提供要求手术并经本人签字的书面报告<sup>*</sup>
> * 手术对象本人提供已告知直系亲属进行手术的相关证明<sup>*</sup>
> * 需由当地公安部门出具手术对象的《无在案犯罪记录证明》<sup>*</sup>
> 
> *代表仅第一性征相关手术适用，院方可能会要求对材料进行公证。

> [!NOTE] 男跨女性别肯定手术一般包括的项目：
> 
> * 第一性征女性化手术
> * 面部女性化手术
> * 喉结缩小手术

> [!NOTE] 女跨男性别肯定手术一般包括的项目：
> 
> * 第一性征男性化手术
> * 面部男性化手术
> * 胸部切除 / 缩小手术

## 嗓音女性化手术

嗓音女性化手术是一种通过手术实现嗓音改变的手段，适用于女性倾向跨性别者。<mark>嗓音手术旨在通过手术调整嗓音，使其更符合认同性别，从而提升自我接纳、缓解性别焦虑并改善心理健康</mark>。

> [!NOTE] 国内目前的嗓音女性化手术医疗资源参考：
> 
> * 首都医科大学附属北京友谊医院耳鼻喉科门诊
> * 南京同仁医院嗓音中心

## 嗓音女性化训练

嗓音女性化训练是一种通过训练实现嗓音改变的手段，适用于女性倾向跨性别者。<mark>嗓音训练旨在通过系统训练调整嗓音，使其更符合认同性别，从而提升自我接纳、缓解性别焦虑并改善心理健康</mark>。

> [!NOTE] 国内目前的嗓音女性化训练医疗资源参考：
> 
> * 首都医科大学附属北京友谊医院耳鼻喉科门诊
> * 上海市长宁区精神卫生中心嗓音言语专病门诊

> [!NOTE] 嗓音女性化训练的主要方式：
> 
> * 基于康复医学的练习方式
> * 基于声乐理论的练习方式

其中，基于康复医学的嗓音训练相较于基于声乐理论的练习对于声带和声道长度这些生理条件的要求较低，主要通过改善受训者发声的声学与感知差异（如基频、音色）实现嗓音女性化。需要注意的是，嗓音训练效果因训练方式和个体生理条件不同而异。

> [!NOTE] 嗓音女性化训练的主要步骤：
> 
> * 通过调整声道和气息将基频提升至超过 175 Hz
> * 通过调整嘴形、呼吸等方式对音色进行修饰
> * 通过对话等练习保证自发言语状态下的嗓音稳定性
